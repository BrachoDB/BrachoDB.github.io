🧩 Sistema de Control de Parqueadero

Sistema web profesional para la gestión integral de parqueaderos, con automatización de cobros y monitoreo de ocupación en tiempo real.

Funcionalidades:

  • Gestión de categorías (Sedán, Camioneta, Motos)
  • Control de capacidad (30 cupos autos / 15 motos)
  • Tarifas dinámicas (por hora, minuto, fracción o día)
  • Generación de tickets detallados de salida
  • Roles de acceso (Administrador y Operario)

🧩 Sistema de Gestión de Cine

Aplicación web diseñada para la administración de cartelera y la venta automatizada de tiquetes de cine, optimizando el proceso de reserva.

Funcionalidades:

  • Gestión dinámica de cartelera de películas
  • Administración de funciones, salas y horarios
  • Generación de códigos QR para entradas digitales
  • Notificaciones por correo mediante Mailjet
  • Sistema de venta y reserva de boletos

🌿 Manjares del Campo - E-commerce

Plataforma diseñada para conectar agricultores colombianos con consumidores finales, eliminando intermediarios y promoviendo el comercio local justo.

Funcionalidades:

  • Catálogo dinámico de productos ("Cosecha de la semana")
  • Pedidos directos automatizados vía WhatsApp
  • Carrito de compras intuitivo para gestión de pedidos
  • Panel administrativo para gestión de productos
  • Diseño 100% responsivo con estética natural
